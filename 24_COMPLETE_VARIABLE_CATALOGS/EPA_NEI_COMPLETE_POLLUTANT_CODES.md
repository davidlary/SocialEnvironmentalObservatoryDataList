## üîç HOW TO USE NEI DATA FOR COUNTY-LEVEL RESEARCH

### **Step 1: Download NEI Data**

**Option A: Pre-Summarized County Data**
- 2020 NEI Supporting Data and Summaries includes county-level files
- Direct link: https://www.epa.gov/air-emissions-inventories/2020-nei-supporting-data-and-summaries

**Option B: NEI Data Retrieval Tool**
- Query by county, pollutant, sector, year
- Export custom datasets

**Option C: Full NEI Downloads**
- Point sources (facilities with lat/lon ‚Üí aggregate to county)
- Nonpoint sources (already county-level)
- On-road mobile (county-level)
- Non-road mobile (county-level)

### **Step 2: County Aggregation Example**

```r
library(tidyverse)
library(sf)

# Example: Aggregate point source emissions to county
# (Point source file has lat/lon, needs spatial join)

point_sources <- read_csv("nei2020_point_sources.csv")
us_counties <- st_read("tl_2020_us_county.shp")

# Convert to spatial
point_sf <- st_as_sf(point_sources,
                     coords = c("longitude", "latitude"),
                     crs = 4326)

# Spatial join to counties
point_with_county <- st_join(point_sf, us_counties)

# Aggregate by county and pollutant
county_emissions <- point_with_county %>%
  st_drop_geometry() %>%
  group_by(GEOID, pollutant_code, year) %>%
  summarize(
    total_emissions_tons = sum(total_emissions, na.rm = TRUE),
    facilities_count = n_distinct(facility_id)
  )

# Focus on criteria pollutants
criteria_pollutants <- c("PM25-PRI", "PM10-PRI", "SO2", "NOX", "VOC", "CO")

county_caps <- county_emissions %>%
  filter(pollutant_code %in% criteria_pollutants)
```

### **Step 3: Link to Health Outcomes**

**County-level exposure metrics:**
- Emissions per capita (tons/population)
- Emissions density (tons/land area)
- Population-weighted exposure (using census blocks)
- Sector-specific contributions (industrial vs. mobile)

## üìö COMPANION EPA AIR DATA SOURCES

### **Ambient Monitoring (Actual Measured Air Quality):**

**EPA Air Quality System (AQS)**
- Monitors actual ambient concentrations
- Complements NEI emissions data
- County-level design values, trends
- Portal: https://www.epa.gov/outdoor-air-quality-data

**Differences:**
- **NEI = Emissions** (what's released)
- **AQS = Concentrations** (what people breathe)
- Health studies should use both!

### **Modeled Concentrations:**

**EPA Air Toxics Screening Assessment (AirToxScreen)**
- Modeled ambient concentrations of HAPs
- Census tract level
- Based on NEI emissions + atmospheric modeling
- Portal: https://www.epa.gov/AirToxScreen

### **Facility-Level Compliance:**

**EPA Enforcement & Compliance History Online (ECHO)**
- Individual facility violations
- Permits and limits
- Enforcement actions
- Portal: https://echo.epa.gov/

## ‚ö†Ô∏è DATA QUALITY NOTES

### **Strengths:**
- Comprehensive source coverage
- Quality-assured by EPA and states
- Long time series (1990-present)
- Multiple pollutant types

### **Limitations:**
- **Triennial updates** (not annual for all sectors)
- **Estimates, not measurements** (except for some large point sources with CEMS)
- **Reporting thresholds** (small sources may not be included)
- **Temporal resolution** (annual totals only, no seasonal/monthly)
- **Spatial resolution** (county-level for many sources, facility-level for point sources)
