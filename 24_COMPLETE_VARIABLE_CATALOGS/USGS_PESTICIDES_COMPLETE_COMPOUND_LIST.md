## üóÇÔ∏è DATA STRUCTURE & COVERAGE

### **Agricultural Use Estimates (EPest):**
- **Spatial:** County-level, conterminous United States (3,109 counties)
- **Temporal:** Annual, 1992-2022 (2018-2022 data releasing 2025)
- **Compounds:** 423 pesticides (herbicides, insecticides, fungicides)
- **Units:** Kilograms of active ingredient applied per county per year
- **Crops:** 50+ major agricultural crops

### **Water Quality Monitoring (NAWQA):**
- **Spatial:** 3,100+ monitoring sites (streams, rivers, groundwater)
- **Temporal:** 1991-present (ongoing)
- **Compounds:** 225 pesticides + degradates in Schedule 2437
- **Detection Limits:** ng/L to ¬µg/L depending on compound
- **Matrices:** Filtered water, suspended sediment

## üîç HOW TO USE USGS PESTICIDE DATA FOR COUNTY-LEVEL RESEARCH

### **Step 1: Download EPest County Data**

```r
library(tidyverse)
library(sf)

# Download EPest data from USGS
# Example: 2013-2017 data (Data Series 1034)

# Method A: Direct download from USGS ScienceBase
url <- "https://www.sciencebase.gov/catalog/file/get/5edc73e582ce3bd58d7be9f3"
temp_file <- tempfile(fileext = ".zip")
download.file(url, temp_file, mode = "wb")

# Extract county-level estimates
unzip(temp_file, exdir = tempdir())
epest_files <- list.files(tempdir(), pattern = "EPest", full.names = TRUE)

# Read all pesticide tables (423 compounds)
epest_data <- epest_files %>%
  map_dfr(~ read_csv(.x, col_types = cols(
    COMPOUND = col_character(),
    YEAR = col_integer(),
    STATE_FIPS_CODE = col_character(),
    COUNTY_FIPS_CODE = col_character(),
    EPEST_LOW_KG = col_double(),
    EPEST_HIGH_KG = col_double()
  )))

# Create full FIPS code
epest_data <- epest_data %>%
  mutate(
    state_fips = str_pad(STATE_FIPS_CODE, 2, pad = "0"),
    county_fips = str_pad(COUNTY_FIPS_CODE, 3, pad = "0"),
    fips = paste0(state_fips, county_fips)
  )
```

### **Step 2: Aggregate Pesticides by Chemical Class**

```r
# Define chemical classes for commonly studied pesticide groups
herbicide_classes <- c("Triazine", "Chloroacetanilide", "Phenoxy",
                       "Sulfonylurea", "Imidazolinone", "Glyphosate")

insecticide_classes <- c("Organophosphate", "Carbamate", "Pyrethroid",
                         "Neonicotinoid", "Organochlorine")

fungicide_classes <- c("Strobilurin", "Triazole", "Dithiocarbamate")

# Aggregate by chemical class for each county-year
class_aggregates <- epest_data %>%
  filter(YEAR == 2017) %>%
  mutate(
    chem_class = case_when(
      COMPOUND %in% c("Atrazine", "Simazine", "Propazine") ~ "Triazine",
      COMPOUND %in% c("Acetochlor", "Alachlor", "Metolachlor", "S-Metolachlor") ~ "Chloroacetanilide",
      COMPOUND %in% c("2,4-D", "MCPA", "Dicamba") ~ "Phenoxy",
      COMPOUND == "Glyphosate" ~ "Glyphosate",
      COMPOUND %in% c("Chlorpyrifos", "Malathion", "Phorate", "Diazinon") ~ "Organophosphate",
      COMPOUND %in% c("Carbaryl", "Carbofuran", "Methomyl") ~ "Carbamate",
      COMPOUND %in% c("Permethrin", "Bifenthrin", "Lambda-Cyhalothrin", "Deltamethrin") ~ "Pyrethroid",
      COMPOUND %in% c("Imidacloprid", "Clothianidin", "Thiamethoxam") ~ "Neonicotinoid",
      COMPOUND %in% c("Azoxystrobin", "Pyraclostrobin", "Trifloxystrobin") ~ "Strobilurin",
      COMPOUND %in% c("Propiconazole", "Tebuconazole", "Prothioconazole") ~ "Triazole",
      TRUE ~ "Other"
    )
  ) %>%
  group_by(fips, YEAR, chem_class) %>%
  summarize(
    total_kg = sum(EPEST_HIGH_KG, na.rm = TRUE),
    n_compounds = n_distinct(COMPOUND)
  ) %>%
  ungroup()
```

### **Step 3: Calculate Population-Normalized Exposure Metrics**

```r
library(tidycensus)

# Get county population from Census
county_pop <- get_acs(
  geography = "county",
  variables = "B01003_001",  # Total population
  year = 2017,
  survey = "acs5"
) %>%
  select(fips = GEOID, population = estimate)

# Join pesticide data with population
pesticide_exposure <- class_aggregates %>%
  left_join(county_pop, by = "fips") %>%
  mutate(
    kg_per_capita = total_kg / population,
    kg_per_1000_people = (total_kg / population) * 1000
  )

# Calculate summary statistics by chemical class
exposure_summary <- pesticide_exposure %>%
  group_by(chem_class) %>%
  summarize(
    mean_kg_per_1000 = mean(kg_per_1000_people, na.rm = TRUE),
    median_kg_per_1000 = median(kg_per_1000_people, na.rm = TRUE),
    counties_with_use = sum(total_kg > 0),
    max_kg_per_1000 = max(kg_per_1000_people, na.rm = TRUE)
  )
```

## ‚ö†Ô∏è DATA QUALITY & LIMITATIONS

### **Strengths:**
- Most comprehensive county-level pesticide use dataset for US
- Covers 423 agricultural pesticides over 30-year period
- Combines USDA surveys, proprietary data (Kynetec, DMRKynetec), state records
- Transparent estimation methods published by USGS
- Free and publicly available

### **Limitations:**
- **Estimates, not measurements:** County use calculated from state/crop surveys, not direct observation
- **Seed treatments excluded (post-2014):** Due to difficulty in estimation
- **Non-agricultural use not included:** Homeowner, structural, industrial uses excluded
- **Uncertainty ranges:** EPest-low vs EPest-high reflect estimation uncertainty
- **Temporal gaps:** Not all years available for all compounds
- **Detection limits:** Water quality data has compound-specific detection limits
- **Discontinued compounds:** Historical compounds may have limited data
- **Spatial resolution:** County-level only, no sub-county information
- **Application timing:** Annual totals only, no seasonal or monthly resolution
