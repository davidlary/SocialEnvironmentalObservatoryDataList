# Infectious Disease & Vector Data Sources

## Where to Find Infectious Disease Variables

Infectious disease data is currently documented in:

### **04_TOXIC_CHEMICALS/**
- **GREEN_PEST_OCC_VECTOR_SOCIAL_COMPREHENSIVE.md**
  - CDC ArboNET: West Nile Virus (2003+)
  - CDC ArboNET: Lyme disease (1992+)
  - Other vector-borne diseases (dengue, Zika, ehrlichiosis)

### **19_ECONOMIC_INDICATORS/**
- **ECONOMIC_FINANCE_HEALTH_FACILITIES_COMPREHENSIVE.md**
  - CDC AtlasPlus: Sexually transmitted infections
  - CDC AtlasPlus: Tuberculosis
  - CDC AtlasPlus: Viral hepatitis (A, B, C)
  - 1996-present

### **02_WATER/**
- **USGS_Water_Quality_Parameters_County_Level_COMPREHENSIVE.md**
  - Waterborne disease pathogens
  - Bacterial contamination (E. coli, total coliforms)
