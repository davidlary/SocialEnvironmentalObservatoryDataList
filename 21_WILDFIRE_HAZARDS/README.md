# Wildfire Hazards Data Sources

## Where to Find Wildfire Variables

Wildfire data is currently documented in:

### **08_INFRASTRUCTURE/**
- **INFRASTRUCTURE_COASTAL_CONTAMINANTS_COMPREHENSIVE.md** ‚≠ê PRIMARY FILE
  - USFS Wildfire Hazard Potential (WHP) - 270m resolution
  - USGS/USFS Wildland-Urban Interface (WUI) - 1990, 2000, 2010, 2020

### **01_AIR_ATMOSPHERE/**
- **ATMOSPHERIC_EMISSIONS_CHEMISTRY_COMPREHENSIVE.md**
  - NOAA HMS wildfire smoke plumes (2005-present, daily)
  - PM2.5 wildfire attribution

### **07_BUILT_ENVIRONMENT/**
- **WILDFIRE_TRAFFIC_WALKABILITY_HAZWASTE_COMPREHENSIVE.md**
  - Wildfire smoke exposure days
  - Smoke PM2.5 estimates
