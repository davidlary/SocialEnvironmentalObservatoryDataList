# Socioeconomic Data Sources

## Where to Find Socioeconomic Variables

Socioeconomic data is currently documented in:

### **19_ECONOMIC_INDICATORS/**
- **BEA_COUNTY_ECONOMIC_DATA_COMPREHENSIVE.md**
  - Personal income (total and by source)
  - GDP by county
  - Earnings by industry

- **CBP_QCEW_COMPREHENSIVE_DOCUMENTATION.md**
  - Employment by industry
  - Unemployment (BLS LAUS)

- **ECONOMIC_FINANCE_HEALTH_FACILITIES_COMPREHENSIVE.md**
  - IRS tax data (income, AGI, deductions)
  - Federal Reserve household debt
  - Bankruptcy filings
  - Building permits
  - Gini inequality index

### **17_SOCIAL_DETERMINANTS/**
- **HUD_HOUSING_COMPREHENSIVE.md**
  - Fair Market Rents (1983+)
  - CHAS housing affordability
  - Cost burden
  - Housing problems

### **18_EDUCATION/**
- **NCES_EDUCATION_DATA_COMPREHENSIVE.md**
  - Educational attainment (Census ACS)
  - School enrollment
