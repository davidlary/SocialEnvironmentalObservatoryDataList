# Flood Hazards Data Sources

## Where to Find Flood Variables

Flood hazard data is currently documented in:

### **08_INFRASTRUCTURE/**
- **INFRASTRUCTURE_COASTAL_CONTAMINANTS_COMPREHENSIVE.md** ‚≠ê PRIMARY FILE
  - NOAA Sea Level Rise (scenarios 0-10 ft)
  - Coastal inundation mapping
  - Population/infrastructure exposure

### **03_LAND_SOIL_GEOLOGY/**
- **HYDROLOGY_WETLANDS_FLOODPLAINS_COMPREHENSIVE.md**
  - FEMA National Flood Hazard Layer (NFHL)
  - 100-year and 500-year floodplains
  - Special Flood Hazard Areas (SFHA)
  - Flood insurance claims (FEMA)

- **NLCD_NOAA_STORMS_COMPREHENSIVE_DOCUMENTATION.md**
  - NOAA Storm Events: flooding events (1996+)
  - Flash floods, river floods, coastal floods
